Session::sql — Execute SQL query

Description

public mysql_xdevapi\SqlStatement mysql_xdevapi\Session::sql ( string $query )

Create a native SQL command. Placeholders are supported using the native "?" syntax. Execute an SQL query.

Warning

This function is currently not documented; only its argument list is available.

Parameters

query

SQL statement to execute.

Return Values

An SqlStatement object.

Examples

Example #1 mysql_xdevapi\Session::sql() example

<?php

$res = $session->sql("insert into test_db.test_table values ('Carlos',9999)")->execute();
$res = $session->sql("select age, age/2 as x from test_db.test_table limit 2")->execute();
$res = $session->sql("select * from $db.test_table where age < ? order by age desc limit ?")->bind(14)->bind(2)->execute();

?>
