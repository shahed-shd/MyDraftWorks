Installing the MongoDB PHP Driver on macOS with Homebrew (deprecated)

» Homebrew 1.5.0 deprecated the » Homebrew/php tap and removed formulae for PHP extensions. Going forward, macOS users should install the driver with PECL. Community forks of the » Homebrew/php tap may still contain formale for installing the driver on various PHP versions.
